# Superstore Sales Analysis

Exploratory data analysis of 9,800 retail orders from the Superstore dataset.

## Key findings
- Strong seasonality: sales peak in November and Q4
- West region leads with $710K in total sales
- Technology is the top category by revenue
- Office Supplies dominate Same Day shipping (60%)

## Tools
Python · Pandas · Matplotlib · Seaborn

- `notebook.ipynb` — Full analysis
